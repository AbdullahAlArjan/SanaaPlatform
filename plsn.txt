القسم الثالث: الصفحات الناقصة المكتشفة من Swagger
🔐 Auth Flow (5 صفحات)
الملف	Endpoints المستخدمة	الوصف
register.html	POST /api/Users	تسجيل مستخدم جديد
otp-verify.html	POST /api/auth/send-otp + POST /api/auth/verify-otp	التحقق من البريد الإلكتروني
freelancer-register.html	POST /api/Freelancers/create-profile	إكمال ملف الصانع
forgot-password.html	POST /api/auth/forgot-password	طلب إعادة تعيين كلمة المرور
reset-password.html	POST /api/auth/reset-password	تعيين كلمة مرور جديدة
🌐 Public Pages (4 صفحات)
الملف	Endpoints	الوصف
index.html	GET /api/Categories + GET /api/Freelancers/search	الصفحة الرئيسية
freelancers.html	GET /api/Freelancers/search?profession&city&serviceId	بحث وتصفية الصنايعيين
freelancer-profile.html	GET /api/Freelancers/{userId} + /portfolio + Reviews	الملف العام للصانع
service-detail.html	GET /api/Services/{id} + GET /api/Reviews/freelancer/{id}	تفاصيل خدمة
👤 Customer Dashboard (8 صفحات)
الملف	Endpoints	الوصف
dashboard-customer.html	بيانات من localStorage + Orders	لوحة تحكم العميل
order-create.html	POST /api/Orders/create	إنشاء طلب جديد
order-track.html	عرض حالة الطلب	متابعة الطلب
payment.html	POST /api/Payments/create-intent + Stripe.js	صفحة الدفع
invoice.html	GET /api/Invoices/order/{orderId}	عرض الفاتورة
review.html	POST /api/Reviews/add	تقييم بعد اكتمال الطلب
profile-settings.html	POST /api/Freelancers/profile-image	إعدادات الحساب
report.html	POST /api/Reports	تقديم بلاغ على مستخدم/خدمة
🛠️ Freelancer Dashboard (3 صفحات)
الملف	Endpoints	الوصف
dashboard-freelancer.html	GET /api/Freelancers/{userId} + Orders	لوحة تحكم الصانع
orders-manage.html	GET /api/Orders/freelancer/{id} + PUT /api/Orders/{orderId}/status	إدارة الطلبات
portfolio-manage.html	GET+POST+DELETE /api/Freelancers/portfolio	إدارة الأعمال السابقة
👑 Admin Panel (6 صفحات)
الملف	Endpoints	الوصف
admin/index.html	GET /api/Admin/dashboard-stats	إحصائيات حقيقية
admin/users.html	GET /api/Users + toggle + delete + deleted list	إدارة المستخدمين
admin/freelancers.html	GET /api/Admin/freelancers/pending + approve/reject	موافقات الصنايعيين
admin/reports.html	GET /api/Reports + PUT /api/Reports/{id}/status	مراجعة البلاغات
admin/categories.html	CRUD كامل /api/Categories	إدارة الفئات
admin/services.html	GET/POST /api/Services	إدارة الخدمات
🔔 Shared Module (1 ملف)
الملف	الوصف
js/notifications.js	SignalR connection للإشعارات الفورية عبر /NotificationHub
