6. AUTHENTICATION MECHANISM
Bearer Token (JWT)

Security scheme: Bearer type API Key
Header: Authorization: Bearer {your_token}
Arabic instruction in schema: "الرجاء إدخال التوكن بهذا الشكل Bearer {your_token}"
Applied globally to all endpoints (security: [{Bearer: []}])
Token refresh endpoint: POST /api/auth/refresh
Token revocation on logout: POST /api/auth/logout
